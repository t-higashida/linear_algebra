\subsection{空間のベクトル}

空間の任意の2点$P,Q$を両端とする線分に$P$から$Q$への向きを指定したものを
\textbf{有向線分}と呼び，$PQ$と書く．
このとき$P$を\textbf{始点}，$Q$を\textbf{終点}という．
有向線分$PQ$と$QP$とは異なる．2つの有向線分$PQ$，$RS$が

\begin{list}{}{}
\item	(1) 4点$P,Q,R,S$が同一直線上にあり，$PQ$をこの直線上で動かして$RS$に重ねることができる．
\item (2) 4辺形$PQSR$がこの順序で平行4辺形になっている．
\end{list}
のいずれかを満たすとき，$PQ$と$RS$は\textbf{同値}であるといい
$PQ \sim RS$と書く．
$PQ \sim RS$であるとき$PQ$と$RS$を同じ類(class)に入れることによって空間のすべての
有向線分を類別することができる．こうして生まれる各同値類を
\textbf{幾何ベクトル}または単に\textbf{ベクトル}という．以下本書では
ベクトル$\boldsymbol{a}, \boldsymbol{b},\boldsymbol{c}, \cdots,
\boldsymbol{x}, \boldsymbol{y},\boldsymbol{z}$などの記号で表すことにする．
特に始点と終点の一致した有向線分の属する類を\textbf{零ベクトル}といい，
$\boldsymbol{0}$で表す．任意のベクトル$\boldsymbol{a}$と任意の点$P$に
対し，有向線分$PQ$が$\boldsymbol{a}$に属するようんた点$Q$がただ1つ存在する．
また，幾何ベクトルを表すときに$\boldsymbol{a}$の任意の代表$PQ$をとって表すことがある．
このときは
\[
	\boldsymbol{a} = \overrightarrow{PQ}
\]
と書き，$\overrightarrow{PQ}$を$P$を始点，$Q$を終点とする\textbf{固定ベクトル}という．

2つのベクトル$\boldsymbol{a},\boldsymbol{b}$を任意の代表によって
\[
	\boldsymbol{a} = \overrightarrow{PQ}, \quad \boldsymbol{b} = \overrightarrow{RS}
\]
と表したとき，$PQ \sim RS$であれば$\boldsymbol{a}$と$\boldsymbol{b}$とは\textbf{相等しい}といい
\[
	\boldsymbol{a} = \boldsymbol{b}
\]
と書く．また線分$PQ$と$RS$とが平行であれば，$\boldsymbol{a}$と$\boldsymbol{b}$とは\textbf{平行}である
といい
\[
	\boldsymbol{a} \mathrel{/\!/} \boldsymbol{b}
\]
と書く．便宜上，零ベクトル$\boldsymbol{0}$はいかなるベクトルに対しても平行であると約束する．
$\boldsymbol{a} = \overrightarrow{PQ}$のとき，線分$PQ$の長さをベクトル
$\boldsymbol{a}$\textbf{の長さ}または\textbf{大きさ}といい，
$|| \boldsymbol{a} ||$で表す．とくに長さ$1$のベクトルを\textbf{単位ベクトル}という．

\[
	\angle AOB = \theta \quad ( 0 \leqq \theta \leqq \pi)
\]
をベクトル$\boldsymbol{a} , \boldsymbol{b}$の$\textbf{b}$の\textbf{なす角}という．
これは始点$O$の選び方によらないことは明らかであろう．特に$\theta = \pi /2$のとき

\[
	\boldsymbol{a} \perp \boldsymbol{b}

\boldsymbol{a},\boldsymbol{b}$の定める並行4辺形の面積を$S$とするとき，次の等式を証明せよ．

\begin{list}{}{}
	\item	(1) $S = || \boldsymbol{a}|||| \boldsymbol{b}|| \sin \theta$.
	\item (2) $S^2 = || \boldsymbol{a}||^2|| \boldsymbol{b}||^2 -(\boldsymbol{a}, \boldsymbol{b})^2$.
\end{list}

\証明
$\boldsymbol{a},\boldsymbol{b}$の間の角を$\theta$とすると，
$\boldsymbol{a},\boldsymbol{b}$の内積は，
\[
	(\boldsymbol{a},\boldsymbol{b}) = ||\boldsymbol{a}||||\boldsymbol{b}|| \cos \theta
\]
ここに$||\boldsymbol{a}||,||\boldsymbol{b}||$は，
\[
	||\boldsymbol{a}||=\sqrt{a_1^2 + a_2^2 },\: \boldsymbol{b} =\sqrt{b_1^2 + b_2^2}
\]
一方，内積は，数ベクトルとしては，次で与えられる（定理2.12の前の記述を参照）
\[
	(\boldsymbol{a},\boldsymbol{b})=a_1b_1 + a_2b_2
\]
ゆえに，
\[
	a_1b_1 + a_2b_2 = \sqrt{a_1^2 + a_2^2 }\sqrt{b_1^2 + b_2^2} \cos \theta
